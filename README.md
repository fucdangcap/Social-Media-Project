# ThreadsLite

Một mạng xã hội tối giản được xây dựng với Next.js 15.

## Công nghệ sử dụng
- **Frontend:** Next.js, React, Tailwind CSS.
- **Backend:** Next.js API Routes.
- **Database:** MongoDB, Mongoose.

## Cách chạy dự án
1. Clone repo về máy.
2. Tạo file `.env.local` và điền `MONGODB_URI`.
3. Chạy lệnh:
   ```bash
   npm install
   npm run dev
